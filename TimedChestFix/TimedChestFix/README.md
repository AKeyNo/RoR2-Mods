# TimedChestFix
Changes the way the Timed Security Chest on Rallypoint Delta is spawned from 10 minutes to 20 minutes.

## Installation
Drop the .dll file into C:\Program Files (x86)\Steam\steamapps\common\Risk of Rain 2\BepInEx\plugins
